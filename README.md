# Serene Hotel Tuần Châu - Demo giao diện website

Bản demo tĩnh của website khách sạn **Serene Hotel** (bến du thuyền Tuần Châu Marina, Hạ Long).
Phong cách luxury, tối ưu đặt phòng trực tiếp. Đa ngôn ngữ **Tiếng Việt / English / 中文**.

## Xem demo
Trang demo chạy trên GitHub Pages: bật tab **Settings → Pages** của repo này (branch `main`, thư mục `/root`).
Link sẽ có dạng: `https://<tài-khoản>.github.io/serene-hotel-demo/`

## Các trang
- Trang chủ (hero, tìm phòng, hạng phòng, tiện ích, trải nghiệm, thư viện, đánh giá, liên hệ)
- Phòng nghỉ, Ưu đãi, Giới thiệu
- Tin tức (blog SEO)
- Chính sách bảo mật

## Ghi chú demo
- Đây là bản **HTML tĩnh** xuất từ WordPress để xem giao diện. Form đặt phòng và blog động sẽ hoạt động đầy đủ trên bản WordPress thật khi lên hosting + tên miền `tuanchauserenehotel.com`.
- Ảnh, giá là bản tham khảo; sẽ thay bằng dữ liệu chính thức từ khách sạn.
